**Lab8_FX: P2P-голосовой чат с Multicast-обнаружением**

Данный проект является восьмой лабораторной работой, посвященной изучению децентрализованных сетей (Peer-to-Peer), работе с протоколами TCP/UDP и реализации потоковой передачи аудиоданных в реальном времени.

**Функциональные возможности**

+ P2P-взаимодействие: каждый узел системы одновременно выступает в роли сервера (ожидание вызова) и клиента (инициация звонка).
+ Голосовая связь: захват аудио (PCM) через TargetDataLine и передача потока по протоколу UDP для обеспечения минимальных задержек.
+ Сигнальный канал: управление сессиями связи (команды CALL_START, CALL_END) через надежное TCP-соединение.
+ Автоматическое обнаружение (LanDiscovery): поиск активных участников в локальной сети с помощью Multicast-рассылки сообщений HELLO без использования центрального сервера.
+ Список контактов: динамическое обновление списка доступных пользователей в GUI на основе механизма Heartbeat (контроль активности узлов каждые 15 секунд).

**Архитектура приложения**

+ Программа построена на многопоточной модели для параллельной обработки различных сетевых задач:
+ Discovery Thread: работа с MulticastSocket для анонсирования себя в сети и прослушивания других участников.
+ Signaling Thread: управление TCP-сервером для приема входящих запросов на звонок.
+ Audio Engine: отдельные потоки для захвата звука с микрофона и воспроизведения полученных UDP-пакетов через SourceDataLine.
+ GUI (JavaFX): визуализация списка «Доступные для звонка», полей ввода параметров и кнопок управления вызовом.
